# Semantic Map

Use this map to turn ordinary objects into 咕咕嘎嘎-style narrative functions. Do not treat the mapping as literal truth; treat it as a reusable comic lens.

## System Lenses

### 石/屎 Lens

Use when the topic involves videos,分享,朋友反应,抽象内容, or low-quality entertainment.

- Make `石` sound valuable: 文献、精选、修为、金石之交、持石。
- Let the hidden底层 be low and embarrassing, but speak about it in a serious tone.
- The main move is雅化: turn `屎` into `石`, then into culture, friendship, mythology, or cultivation.

### 朋友/好感度 Lens

Use when the topic involves聊天,邀约,冷场,社交失败, or被朋友吐槽.

- Friend is both loved and harmed.
- Refusal becomes傲娇; silence becomes隐藏剧情; mockery becomes亲密证明.
- The narrator usually admits, indirectly, that they are causing the problem.

### 嘎啦game Lens

Use when the topic can become a relationship route.

- Object becomes可攻略角色.
- Progress becomes好感度、剧情、选项、特殊CG、结局.
- The narrator is both player and被攻略对象.

### 地球online Lens

Use when the topic is a system complaint: weather, age, sleep, exams, social life, bugs, missing features.

- Reality is a badly balanced MMO.
- Bad luck becomes patch notes,版本更新,策划恶意,锁敌,属性倒扣.
- Desired fixes are好友系统、读档、挂机、跳过剧情、传送、托管.

### 古文/文豪/修仙/二游 Lens

Use when the topic benefits from high-low contrast.

- Ancient lines become modern complaints.
- 文豪 become角色卡,命座,豪意值,技能组.
- 修仙 terms explain weak self-control: 灵根、道心、渡劫、本命法器、飞升.

## Target Selection

Choose one main target per short script:

- Relationship target: 朋友、同学、亲戚、老师、网友。
- Object target: 手机、床、被子、试卷、外卖、闹钟。
- System target: 地球online、学校、公司、天气、节日。
- Self target: 自制力、情商、脑子、身体功能、睡眠。

Avoid stacking too many targets. One primary system plus one target is usually cleaner than five references fighting each other.
