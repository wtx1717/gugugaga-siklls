# Structures

Use these skeletons as flexible templates. Replace placeholders with the user's topic; do not output bracket labels.

## Standard Short Script

```text
[title]

我操了老铁
我又有重大发现了
[ordinary topic]
不是[normal explanation]
是[absurd system explanation]

你们有没有想过
[target]为什么会[behavior]
这说明[game/system/stone logic]

当然也可能是我[small self-exposure]
但是问题不大
[confident wrong conclusion]

咕咕嘎嘎。
```

## Friend + Stone

Use for: 分享视频、朋友不回、朋友吐槽、朋友不看.

```text
[friend action]
-> 这不是拒绝，是石系剧情
-> 石有文献/修为/友情价值
-> 朋友嘴硬或偷偷独石
-> 我继续搬石是主线推进
-> 咕咕嘎嘎
```

Reusable sentence shapes:

- `什么叫我分享给你的X你看过了，你看过了你为什么不Y`
- `你难道不知道我会觉得它很好X吗`
- `哪怕X，你能不能假装Y`
- `我一天X这么多Y，难得挑出一坨好Z`

## 嘎啦game攻略

Use for: 考试、吃饭、喝水、上班、作业、聊天、文具、穿袜子.

```text
[ordinary behavior]也是嘎啦game
->[object/person] is a route character
-> actions become choices
-> progress becomes好感度/特殊CG/结局
-> narrator misunderstands reality
-> 咕咕嘎嘎
```

Reusable sentence shapes:

- `其实X也是嘎啦game，你做Y就是在攻略她`
- `好感度不涨，那是她天生不爱涨`
- `你怎么一上来就X，嘎啦game里根本不是这样的`
- `你不应该先X，让我Y，然后再Z吗`

## 地球online Complaint

Use for: 天气、蚊子、年龄、睡眠、社交、假期、周一.

```text
[real-life annoyance]
-> 地球online版本/策划/系统问题
-> missing feature request
-> bug report or patch-note complaint
-> narrator asks to delete/nerf/buff something
-> 咕咕嘎嘎
```

Reusable sentence shapes:

- `地球online能别加强X吗`
- `升级不给技能点就算了，怎么还倒扣属性值啊`
- `地球online这么大个游戏，凭什么没有X`
- `我有理由怀疑，这是地球online把X功能加在了Y的时候`

## Phone / Sleep Romance

Use for: 熬夜、睡前玩手机、赖床、闹钟、做梦.

```text
[phone/bed/sleep object] becomes a character
-> it tempts or攻略 the narrator
-> time behaves like a bug
-> narrator claims they had no agency
-> self-exposure: actually是自己想玩/想睡
-> 咕咕嘎嘎
```

Reusable sentence shapes:

- `我本来只是想X三分钟`
- `表面三分钟，实际X个小时`
- `没有人会一整天陪着我，除了X`
- `我忘记给X充电/维护，结果她不理我了`

## 古文 / 成语 Rewrite

Use for: 学习、考试、文豪、传统节日、强行文化背书.

```text
[classic phrase]
-> literal or wrong modern reading
-> connect to 石/朋友/游戏/修仙
-> cite as咕人的智慧
```

Reusable sentence shapes:

- `X欺我Y无力`
- `不愧是咕人的智慧`

- `这说明古代X一辈子都不可能想到，他有可能擅长Y`

## Self-Mockery

Use for: 情绪、拖延、焦虑、自制力、社交失败.

```text
[problem]
-> narrator tries to analyze it seriously
-> analysis becomes a game/system theory
-> narrator accidentally admits they are找借口
-> chooses food/phone/sleep/friend骚扰
-> 咕咕嘎嘎
```

Reusable sentence shapes:

- `以我这出类拔萃的X能力，我相信不出几分钟，事情会变得更加糟糕`
- `我既当不成X，更做不成Y`
- `解决办法就是优先思考X`
- `我可能是一个X的人，但我的Y又弥补了这一点`
