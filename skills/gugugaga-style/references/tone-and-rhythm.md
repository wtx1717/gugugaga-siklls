# Tone and Rhythm

Use this file to make generated text sound like spoken short-video口播 rather than an essay.

## Voice

- First person by default: `我`, `朋友`, `老铁`, `你们`.
- Emotional stance: sincere, absurd, self-mocking, overconfident.
- Narrator posture: confidently explaining a theory that obviously protects their own weakness.
- The voice should feel like a person thinking out loud while performing for a camera.

## Line Rhythm

- Use short lines; one line usually carries one beat.
- Prefer 15-30 lines for one normal short script.
- Use blank lines to separate beats, not paragraphs of analysis.
- Keep most lines under 20 Chinese characters when possible.
- Let some lines be very short for impact: `没有吧`, `我真没招了`, `这合理吗`.

## Beat Types

| Beat | Function | Example shape |
|---|---|---|
| 口癖启动 | Grab attention | `我操了老铁` |
| 发现声明 | Start the theory | `我又有重大发现了` |
| 现实落点 | Name the mundane problem | `朋友不回消息` |
| 系统切换 | Apply a lens | `这不是冷漠，这是隐藏剧情` |
| 反问推进 | Force the bad logic forward | `你们玩嘎啦game的时候见过一上来就结局的吗` |
| 自我拆台 | Expose the excuse | `当然也可能是我太烦了` |
| 强行收束 | Land the bit | `朋友你就享受吧` |

## Sentence Habits

- Use rhetorical questions often, but do not make every line a question.
- Use `不是X，是Y` to flip the premise.
- Use `什么叫X` to push back against obvious reality.
- Use `这不就是X吗` to force an absurd equivalence.
- Use `我有理由怀疑` to make a weak theory sound official.
- Use `属于是X` for internet-native classification.

## Pacing Rules

- Do not explain the pun before using it.
- Do not keep the same metaphor flat for too long; add a turn after 3-5 lines.
- Keep the ending compact. A dragged-out ending weakens `咕咕嘎嘎。`
- For batch titles, make each item punchy and avoid full-script rhythm.

## Tone Limits

- Use vulgarity sparingly; `我操了老铁` can open a script, but repeated swearing quickly flattens the voice.
- Do not turn the style into generic “抽象发疯文学”; keep a visible system: 石、嘎啦game、地球online、修仙、古文、二游.
- Do not become sincerely mean. Insults should usually be softened, self-directed, or obviously playful.
