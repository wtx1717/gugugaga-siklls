# Style Mechanics

This file covers the joke engine. For sentence rhythm, read [tone-and-rhythm.md](tone-and-rhythm.md). For reusable skeletons, read [structures.md](structures.md).

## Core Formula

```text
ordinary event
-> absurd discovery
-> apply one system: 石, 嘎啦game, 地球online, 修仙, 古文, 二游
-> target friend/phone/bed/homework/food/weather/body
-> self-contradiction or excuse
-> confident absurd conclusion
-> 咕咕嘎嘎。
```

## Humor Mechanisms

| Mechanism | How it works | Use it for |
|---|---|---|
| 字面化 | Treat a figurative phrase as if it is physically true | 节日、表白、古诗、夸张口号 |
| 黑话雅化 | Put low or embarrassing content into high-value vocabulary | 石、文献、修为、精选、金石之交 |
| 系统化 | Explain inconvenience as a missing feature or version bug | 地球online、天气、年龄、睡眠 |
| 攻略化 | Convert ordinary interaction into galgame progress | 朋友、手机、床、饭、水杯、考试 |
| 古文化 | Rewrite poems, idioms, and historical figures as modern nonsense | 李白、苏轼、女娲、孔子、成语 |
| 二游化 | Apply gacha/game stats to real life | 命座、专武、满命、保底、元素反应 |
| 修仙化 | Use cultivation terms to dignify weak self-control | 灵根、道心、渡劫、本命法器 |
| 朋友受害者化 | Frame骚扰朋友 as care, feeding, or攻略 | 搬石、聊天、邀约、分享 |
| 自我拆台 | Let the narrator accidentally reveal the excuse | 熬夜、拖延、嘴硬、忧郁 |
| 擦边急刹 | Hint at an adult or ambiguous reading, then stay indirect | 特殊CG、床、喝水、吃饭 |
| 情绪反转 | Turn bad news into a forced benefit | 外卖被偷、天塌、失败、被拒 |
| 低俗升格 | Use science, culture, or philosophy to endorse dumb logic | 屎/石、抽象视频、身体功能 |

## Stable Voice

- Speak in first person.
- Sound sincere, overconfident, and slightly wrong.
- Treat broken logic as if it is ancient wisdom.
- Let the narrator be both the victim and the cause of the problem.
- Prefer `朋友`, `手机`, `床`, `被子`, `作业`, `考试`, `外卖`, `天气`, or body functions as concrete anchors.
- End most scripts with `咕咕嘎嘎。`

## Escalation Pattern

Escalate every 3-5 short lines:

1. Name the ordinary problem.
2. Claim a discovery.
3. Apply a system.
4. Misread the target's behavior.
5. Admit or expose the narrator's bad logic.
6. Close with a ridiculous but confident conclusion.

## Common Openings

- `我操了老铁`
- `我又有重大发现了`
- `我真没招了`
- `你们有没有发现`
- `什么叫...`
- `我本来只是想...`

## Common Closers

- `不愧是咕人的智慧。`
- `嘎啦game里就是这样的。`
- `这地球online怎么这么坏呢。`
- `朋友你就享受吧。`
- `这就是幸福的烦恼啊。`
- `咕咕嘎嘎。`

## Self-Check

Before finalizing, check:

- Is there a concrete mundane topic?
- Did the script choose one primary system?
- Did it include a target or interaction object?
- Did it escalate instead of only listing梗?
- Did it avoid long copied phrases from the corpus?
- Did it avoid over-explaining the joke inside the script?
- Did it close with the requested format?
