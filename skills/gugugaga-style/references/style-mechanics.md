# Style Mechanics

This file covers the joke engine. For sentence rhythm, read `tone-and-rhythm.md`; for reusable skeletons, read `structures.md`; for extracted sentence shapes, read `transferable-sentence-shapes.md`.

Core flow: ordinary event -> absurd discovery -> apply one system -> target a concrete object or person -> self-contradiction or excuse -> confident absurd conclusion -> 咕咕嘎嘎。
