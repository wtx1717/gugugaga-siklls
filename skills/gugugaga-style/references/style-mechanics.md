# Style Mechanics

## Stable Voice

- Use first-person oral delivery.
- Use short lines, fast jumps, and rhetorical questions.
- Start with exaggerated certainty.
- Treat broken logic as if it is ancient wisdom.
- Use friend, phone, bed, homework, food, weather, or body functions as interaction targets.
- Usually end with `咕咕嘎嘎。`

## Core Formula

```text
ordinary event
-> absurd discovery
-> apply one system: 石, 嘎啦game, 地球online, 修仙, 古文, 二游
-> target friend/phone/bed/homework/food
-> self-contradiction or excuse
-> confident absurd conclusion
-> 咕咕嘎嘎。
```

## Common Openings

- `我操了老铁`
- `我又有重大发现了`
- `我真没招了`
- `你们有没有发现`
- `什么叫...`

## Common Closers

- `不愧是咕人的智慧。`
- `嘎啦game里就是这样的。`
- `这地球online怎么这么坏呢。`
- `朋友你就享受吧。`
- `咕咕嘎嘎。`

## Generation Patterns

### Friend + Stone

```text
friend ignores video
-> video is stone
-> stone is friendship evidence / cultivation material / ancient reference
-> friend is嘴硬 or eating独石
-> share more stones
```

### Earth Online

```text
real-life bug
-> blame 地球online design
-> demand feature: save/load, friend list, 挂机, time skip, patch notes
-> accuse planner/system
```

### Galgame

```text
ordinary object
-> 可攻略角色
-> 好感度 / 攻略线 / 特殊CG
-> phone/bed/food/homework becomes romantic target
-> narrator is both player and victim
```

### Classical Text Rewrite

```text
classic poem/idiom
-> literal or wrong modern translation
-> connect to game, stone, friend, or phone
-> cite it as 咕人的智慧
```

### Self-Mockery

```text
serious worry
-> narrator thinks deeply
-> realizes they are 傻福
-> chooses food/phone/sleep instead
-> says they only want the viewer/friend to be happy
```

## Pacing Rules

- Use one main metaphor system per script; two is acceptable, more can become muddy.
- Escalate every 3-5 lines.
- Let the narrator sound sincere while saying obviously broken logic.
- Use parentheses sparingly for self-exposure: `（正在寻找借口）`.
- Avoid explaining puns inside the script.

## Self-Check

Before finalizing, check:

- Does the request begin with `以咕咕嘎嘎的方式`?
- Is there a concrete mundane topic?
- Did the script use at least one core system?
- Did it include friend/phone/bed/stone or another target?
- Did it avoid long copied phrases from the corpus?
- Did it end with `咕咕嘎嘎。`?
